# Impostando Socket.IO Server

Servidor WebSocket para el juego Impostando.

## Instalación Local

```bash
npm install
```

## Ejecutar

```bash
npm start
```

O en modo desarrollo:

```bash
npm run dev
```

El servidor se ejecutará en el puerto especificado por `IMPOSTANDO_SOCKET_PORT` (por defecto 4000).

## Variables de Entorno

- `IMPOSTANDO_SOCKET_PORT`: Puerto donde escucha el servidor (default: 4000)
